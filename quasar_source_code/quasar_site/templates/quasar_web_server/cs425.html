1.1

If an equation involves the derivative of one variable with respect to another, then the former is called a dependent variable and the latter an independent variable. Thus, in the equation `(d^2x)/(dt^2)+a(dx/dt)+kx=0`, t is the independent variable and x is the dependent variable.


The order is the order of the highest-order derivatives present in the equation.


A linear differential equation is one in which the dependent variable y is and its derivatives appear in additive combinations of their powers. A differential equation is linear if it has the format `a_n(x)((d^ny)/(dx^n))+a_(n-1)(x)((d^(n-1)y)/(dxn^(n-1)))+...+a_1(x)dy/dx+a_0(x)y=F(x)`, where `a_n(x)`,`a_(n-1)(x)`,...,`a_0(x)` and `F(x)` depend only on the independent variable x.
